## The [`typeof`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/typeof) operator

| Type             | Result                              |
| ---------------- | ----------------------------------- |
| Undefined        | `"undefined"`                       |
| Null             | `"object"` (for historical reasons) |
| Boolean          | `"boolean"`                         |
| Number           | `"number"`                          |
| String           | `"string"`                          |
| Symbol           | `"symbol"`                          |
| Function Object  | `"function"`                        |
| Any Other Object | `"object"`                          |
